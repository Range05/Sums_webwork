Web design

Web design encompasses many different skills and disciplines in the preduction and maintenance of websites. the different areas of web design include web graphic web graphic design (UI design); authoring, including standardised code and proprietary software; user experience design(UX design); andsearch engine optimization. Often many individual will work in teams covering different aspects of the design process relating to the front-end (client side) design of website including writing markup. Web design partially overlaps web engineering in the broader scope of web development. Web designers are expected to have an awareness of usability and be up to date with web accessibillity guidelines
