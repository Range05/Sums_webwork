Perusahaan Startup XYZ

Perusahaan startup XYZ merupakan perusahaan khayalan yang hanya ada pada modul pembelajaran pratikum di lab perancangan web. Hal ini bertujuan untuk mengetes kemampuan mahasiswa dalam membuat sebuah web yang dapat membantu pengguna dalam merasakan produk yang ada pada kehidupan sehari-hari.

Perlu diketahui terkait matakuliah ini membahas tentang membangun aspek visual dan interaktif dari aplikasi web dengan mempelajari HTML, CSS, Javascript, dan Git untuk pembuatan sebuah web dari sisi front end.
